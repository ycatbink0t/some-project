## Description

My fancy crypto profit calculator MVP

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run build
$ npm run start:prod
```

## API docs

```bash
$ curl localhost:3000/api
```

Important! You should run app in production mode if you want to see correct API specification on route /api

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
